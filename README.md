# My job interview algorithm solutions
Here I gather problems resolution to algorithm challenges found in job interviews

### Summary
- secret_message_solution: A challenge from Data Annotation enterprise. The problem consists of reading a table with a set of coordinates (X, Y) which store special characters. A sub-set of specific coordinates represents a letter and all of them in the table represent the message to be revealed. The table bellow store coordinates and characters to F letter:

| x-coordinate | Character | y-coordinate |
| --- | --- | --- |
| 0 | █ | 0 |
| 0 | █ | 1 |
| 0 | █ | 2 |
| 1 | ▀ | 1 |
| 1 | ▀ | 2 |
| 2 | ▀ | 1 |
| 2 | ▀ | 2 |
| 3 | ▀ | 2 |
